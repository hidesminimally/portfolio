Ashby Village

Project Lead and Web Developer

Ashby Village is a nonprofit that connects members with each other and with the resources they need to remain active and independent. Ashby Village needed a tool to increase village communication and event generation. During the fall semester of 2015, I led a team of Blueprint designers and developers to conceptualize and build Ashby Village Square.


With Ashby Village Square, we set out to build an application that is both easy and fun for villagers to use. We first conducted several user interviews to understand core problems that the villagers were facing. Then we tailored the user interface to address common problems we learned through the interviews, and the flow was tailored to fit the existing village hierarchy and organizational structure.


Ashby Village Square was built with Ruby on Rails. Feel free to ask me about feature development, design decisions, or the application architecture! Click here and here to read the Medium articles on our work with Ashby Village. Check out the public repo here!
